Eingabedialoge mit PopUp.java

Damit eine Anwenderin eine Eingabe in unseren Programmen machen kann bedienen wir uns einer zusätzlichen externen Datei.

Vorbereitung

  • Lade die Datei PopUp.java herunter und entpacke sie.
  • Ziehe die Datei in das Processingfenster. Dein Programm sollte dann zwei Reiter besitzen.
  • Nun kannst du die Methoden ("Befehle") von PopUp.java in deinem Programm verwenden.
  • Du musst die Datei in jedes Programm einbinden, in dem du PopUp.java verwenden möchtest.

Anwendung

PopUp.java stellt verschiedene Methoden zu Verfügung, um entweder einen String, int oder float einzulesen.

Als Argument erhält jede Methode den Text, den der/die Benutzer*in im Eingabefenster zu sehen bekommt.


//PopUp.readLine() liest einen String ein.
String product = PopUp.readLine("Was möchtest du kaufen?");

//PopUp.readInt() liest einen Integer (ganze Zahl) ein.
int kg = PopUp.readInt("Wie viele kg kaufst du?");

//PopUp.readFloat() liest einen float (Kommazahl) ein.
float price = PopUp.readFloat("Wie viel kostet ein kg?");

print(kg+" kg "+product+" kosten "+kg*price+" €.");
