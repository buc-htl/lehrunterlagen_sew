Lesen und Schreiben von Textdateien.

Die Details zum Lesen und Schreiben von Dateien werden wir in der 2. Klasse behandeln. Aber auch ohne die Details zu verstehen, kannst du den angeführten Code schon jetzt verwenden um Textdateien zu lesen und schreiben.

Textdatei lesen


    /**
     *  Liest alle Zeilen von einer Textdatei.
     *
     * @param filename  Der Name der Datei
     * @return Array mit allen Zeilen
     */
    public static String[] readAllLinesFromFile(String filename) {
        try {
            List<String> lines = Files.readAllLines(Paths.get(filename), StandardCharsets.UTF_8);
            return lines.toArray(new String[0]);
        } catch (IOException io) {
            System.out.println("Datei konnte leider nicht gelesen werden ("+io.getMessage()+")");
        }
        return null;
    }

    public static void main(String[] args) {
        //Achtung! Der Pfad startet im Ordner des Projekts. Die Datei muss also im Projektordner liegen.
        String[] lines = readAllLinesFromFile("test.txt");
        System.out.println(Arrays.toString(lines));

        //Diese Datei liegt in einem Ordner "resources" innerhalb des Projektordners (gleiche Ebene wie z.B. der Ordner "src").
        //Es ist empfehlenswert alle Dateien in einem eigenen Ordner zu organisieren.
        String[] lines = readAllLinesFromFile("resources/test.txt");
    }

Textdatei schreiben


        public static void main(String[] args) {
            String[] output = {"Zeile 1", "Zeile 2", "Zeile 3"};
            writeAllLinesToFile("output.txt",output);
        }
    
        /**
         * Schreibt den Inhalt von lines zeilenweise in das File.
         * Bestehender Inhalt wird überschrieben.
         * @param filename  Der Dateiname
         * @param lines     Array mit den Zeilen.
         */
        public static void writeAllLinesToFile(String filename, String[] lines) {
            List<String> content=Arrays.asList(lines);
            try {
                Files.write(Paths.get(filename),content);
            } catch (IOException io) {
                System.out.println("Datei konnte leider nicht geschrieben werden ("+io.getMessage()+")");
            }
        }
