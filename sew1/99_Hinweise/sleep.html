Programm pausieren

Folgendes Codebeispiel zeigt, wie man das Programm für eine gewisse Anzahl von Sekunden pausieren kann.


        while (true){
            System.out.println("Hallo");
           
            //pausiert das Programm für 2 Sekunden
            try {
                TimeUnit.SECONDS.sleep(2);
            } catch (InterruptedException e) {
                e.printStackTrace();
            }
        }
