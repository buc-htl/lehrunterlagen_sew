Animationen mittels draw()

Processing kennt zwei fix vorgegebene Funktionen:

  • setup(): Wird einmalig zum Programmstart aufgerufen.
  • draw(): Wird nach setup() in regelmäßigen Intervallen immer wieder aufgerufen.

Mittels draw() können wir sehr einfach Animationen erstellen. Animationen funktionieren nach dem Prinzip des Daumenkinos. Man zeichnet die zu animiertende Figur immer ein wenig versetzt. Ändert sich die Position der Figur schnell genug, wirkt das auf unser Auge, als würde sich die Figur bewegen.

Processing ruft draw() automatisch mehrmals pro Sekunde auf. Zeichnet man in draw() die zu animierende Figur bei jedem Aufruf also leicht versetzt, wirkt das wie eine Bewegung.

Bevor die Figur an der neuen Position gezeichnet wird, muss die alte Figur von der Zeichenfläche gelöscht werden. Dies geschieht am einfachsten indem die Hintergrundfarbe mittels background() erneut gesetzt wird.

Da draw() keine Parameter besitzt, muss die Position der Figur in globalen Variablen gespeichert werden.

Das folgende Programm liefert dir ein sehr einfaches Beispiel für eine Animation.


int x = 20; //Startposition
int dx = 2; //Veränderung in jedem Frame

void setup() {
 size(400,400);
 background(51);
}

void draw() {
  background(51);     //"lösche" die Zeichenfläche
  rect(x,40,20,20);   //zeichne die Figur
  x = x+ dx;            //verschiebe die Position
}
